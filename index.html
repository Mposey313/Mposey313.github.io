Things Needed for a starter PC

  1. Case
  2. Motherboard
  3. GPU [Graphics Card] (if no integrated GPU)
  4. RAM [Memory]
  5. Storage Device (SSD, NVME SSD, HDD)
  6. Cooling (CPU, Chassis)
  7. PSU [Power Supply Unit]
  8. Display device, Monitor
  9. Operating System [OS]
  10. Input Devices, Mouse, Keyboard
